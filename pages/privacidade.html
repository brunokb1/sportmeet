Privacidade
🔒
Coleta de dados
O SportMeet é um aplicativo de demonstração e não coleta dados pessoais reais. Todos os dados são armazenados localmente no seu dispositivo.
📱
Armazenamento local
Suas preferências e eventos são salvos no localStorage do navegador. Nenhuma informação é enviada para servidores externos.
🍪
Cookies
Este app não utiliza cookies de rastreamento. Apenas o localStorage é utilizado para persistência de dados.
🗑️
Exclusão de dados
Para apagar todos os dados, acesse Perfil → Reiniciar demonstração. Isso remove todos os dados locais permanentemente.
